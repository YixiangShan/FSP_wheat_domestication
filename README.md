# Model Simulation for Wheat Growth (GroIMP)

This repository contains the source code and model parameters for the research paper. The model is built on the **GroIMP** (Growth Grammar Interactive Modeling Platform).

## 🛠 Prerequisites

Before running the model, you must have **GroIMP** installed:
* **Download/Install:** [https://gitlab.com/grogra/groimp](https://gitlab.com/grogra/groimp)
* **Official Website:** [www.grogra.de](http://www.grogra.de)
